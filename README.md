RF401 Contract Timeline Tool

A web-based timeline management tool for Tennessee RF401 Purchase & Sale Agreement contracts. Built for real estate agents who need to track deadlines, key dates, tasks, and financial terms across multiple deals.

Features

Deal Dashboard — Create and manage multiple real estate deals from a central hub
Manual Entry — Input key dates, deadline periods, and financial terms from the RF401 contract
Auto-Generated Timelines — Automatically calculates inspection deadlines, financing contingencies, closing dates, and more based on the Binding Agreement Date
Calendar View — Visual monthly calendar with all events and deadlines
Task Tracking — Categorized task checklists (Inspection, Financing, Closing) with status toggles
Deal Info Cards — Contract terms and financial details at a glance
PDF Deal Card Export — Generate a branded, portable PDF summary of any deal
Calendar Export — ICS file download and Google Calendar integration for all events
Cloud Sync — Deals sync across devices via Firebase so you can access them from desktop or mobile
Admin Panel — Manage all users and deals across accounts (admin access only)
Archive & Delete — Archive completed deals or permanently remove them
Offset Overrides — Adjust individual deadline calculations without re-entering the whole form
Tech Stack

Static site hosted on GitHub Pages — no backend server required. Pure vanilla JavaScript with Firebase Firestore for cloud storage and anonymous authentication.
